NonStop Explorer demonstrates how a web or mobile application can exchange messages with an HP NonStop Server Pathway application using LightWave Server from NuWave Technologies. The application retrieves system information from an HP NonStop Server Pathway application via LightWave Server using a User-defined REST API.

LightWave Server is client / server middleware that provides access to NonStop Server standalone process, Pathway serverclass, TMF, and Enscribe services. Distributed applications can access these services using a simple RESTful API. For information on LightWave Server visit the NuWave Technologies Documentation Center

An Android version of this application is available at the Google Play Store.

A web version of this application is available at explorer.nuwavetech.io.

The source code for this application is available on Github.
